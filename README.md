# AI Infrastructure Platform on Kubernetes

End-to-end MLOps setup for deploying and monitoring AI models.

## Goal
Deploy a scalable AI inference service (e.g. RAG or simple LLM) using Kubernetes, with full observability and security.

## Tech Stack
- Docker
- Kubernetes (EKS or kind)
- Prometheus + Grafana
- Terraform (IaC)
- GitHub Actions (CI/CD)
- Python + FastAPI (or Flask)
